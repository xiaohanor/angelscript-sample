# UE-Angelscript Skill

An Agent Skill for AI Coding that provides comprehensive guidance for developing games with Unreal Engine Angelscript.

## What is UE-Angelscript?

Unreal Engine Angelscript is a scripting language plugin for UE5 developed by Hazelight (creators of *It Takes Two* and *Split Fiction*). It enables:

- **Rapid Iteration**: Instant hot-reload without recompiling
- **Improved Cooperation**: Programmers and designers use the same tools
- **Performance**: Approaches C++ performance with transpiled scripts

## Skill Contents

This skill covers:

1. **Installation & Setup** - Getting started with the plugin
2. **Core Concepts** - Syntax, types, and basic structure
3. **Functions & Events** - Declaring and using functions
4. **Properties** - UPROPERTY specifiers and accessors
5. **Actors & Components** - Creating and managing game objects
6. **Networking** - RPCs and replicated properties
7. **Subsystems** - Game architecture patterns
8. **Editor Scripting** - Editor automation tools
9. **Best Practices** - Performance tips and patterns

## Usage

This skill is designed for AI assistants helping with:

- Writing Angelscript gameplay code
- Converting Blueprint to Angelscript
- Converting C++ to Angelscript
- Debugging Angelscript issues
- Architecture guidance for UE-AS projects
